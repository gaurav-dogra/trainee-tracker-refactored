Sparta Global Training Final Project: The Trainee Tracker Introduction This project revolves around the creation of self evaluation software for trainees to use throughout the course, with their trainers able to view their data. This is currently handled by Sparta via the use of tracker spreadsheets updated at the end of each week. The project has room for additional functionality, such as summary reports and graphs that track weekly scores in order to aid trainers. The main focus is on the delivery of a functioning tracker system, seeded with dummy course and trainee data with web forms being added after this setup stage to manage the system. The Projects Database system The tracker system has a database which contains tables for Trainers, Trainees, and Trackers.
